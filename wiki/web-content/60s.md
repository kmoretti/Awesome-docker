# 60s 部署与更新教程

## 这个项目是做什么的

`60s` 是一个每日资讯类项目，常见用途是展示“60 秒读懂世界”这类内容页面。

## 这篇教程适合谁

- 想快速搭一个简单内容页的人
- 想练习最基础单容器部署的人
- 想先从简单项目开始上手 Docker 的人

## 你需要准备什么

- 已安装 Docker 的服务器
- 建议目录：`/opt/docker/60s`
- 端口 `4399`
- 可选域名：`60s.your-domain.com`

## 部署前先检查

```bash
docker -v
docker compose version
```

## 建议目录结构

```bash
/opt/docker/60s/
└── compose.yml
```

## 第一步：创建项目目录

```bash
mkdir -p /opt/docker/60s
cd /opt/docker/60s
```

## 第二步：创建 compose 文件

```yaml
services:
  news60s:
    image: vikiboss/60s:latest
    container_name: 60s
    restart: unless-stopped
    ports:
      - "4399:4399"
```

## 第三步：检查配置

```bash
docker compose config
```

## 第四步：启动

```bash
docker compose up -d
```

## 第五步：访问

```text
http://服务器IP:4399
```

如果做了反向代理，也可以访问：

```text
https://60s.your-domain.com
```

## 数据目录

这个项目通常不依赖必须持久化的数据目录。
如果后续你自己加了配置文件或缓存目录，可以再补充挂载。

## 如何更新

```bash
cd /opt/docker/60s
docker compose pull
docker compose up -d
```

## 如何查看日志

```bash
docker logs -f 60s
```

## 常见问题

### 1. 页面打不开

先确认：

- 容器是否已经启动
- `4399` 端口是否已开放
- 安全组和防火墙是否放行

### 2. 我想改成域名访问

推荐用 `Nginx Proxy Manager` 把域名转发到 `4399` 端口。
